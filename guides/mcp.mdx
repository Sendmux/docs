The Sendmux documentation MCP server lets AI tools search Sendmux guides and API references from your editor or assistant.

## Server URL

```text
https://docs.sendmux.ai/mcp
```

The server exposes a `search_sendmux` tool for documentation search.

## Connect Claude

<Steps>
  <Step title="Open settings">
    Open **Settings** and choose **Connectors**.
  </Step>
  <Step title="Add connector">
    Select **Add Connector**.
  </Step>
  <Step title="Enter the URL">
    Use `https://docs.sendmux.ai/mcp` as the server URL.
  </Step>
</Steps>

## Connect Claude Code

```bash
claude mcp add sendmux-docs --transport http https://docs.sendmux.ai/mcp
```

## Connect Cursor

Add this server to your MCP configuration:

```json
{
  "mcpServers": {
    "sendmux-docs": {
      "url": "https://docs.sendmux.ai/mcp"
    }
  }
}
```

## Connect VS Code

Add this server to your settings:

```json
{
  "mcp": {
    "servers": {
      "sendmux-docs": {
        "type": "http",
        "url": "https://docs.sendmux.ai/mcp"
      }
    }
  }
}
```

## Rate limits

| Scope | Limit |
| --- | --- |
| Per user IP | 5,000 requests per hour |
| Per documentation site | 10,000 requests per hour |
